## Summary

These flow diagrams represent the core game mechanics systems:

1. **Evolution System**: Player-driven mutation paths with prerequisites, incompatibilities, and visual feedback
2. **Combat System**: Turn-based with hit detection, damage types, status effects, and enemy AI
3. **Resource System**: Multi-currency economy with rarity tiers and biome modifiers
4. **Progression System**: Per-run and meta-progression with milestones and permanent upgrades
5. **Run-Ending**: Clear victory/defeat conditions with informative feedback

For implementation details, see the corresponding code stubs in `src/systems/` and balancing tables in `src/data/balancing/`.
